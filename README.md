# Shopping Cart

A simple and interactive **Shopping Cart** web application built with **HTML, CSS, and JavaScript**. This app allows users to add multiple items to a cart, see calculations for subtotal, taxes, and total amount dynamically.

---

## 🛒 Features

- Display a catalog of items (burgers) with **name** and **price**.
- Add items to the cart by clicking the **Add to Cart** button.
- Dynamic cart updates:
  - Quantity of each item
  - Subtotal
  - Taxes (5% per item)
  - Total amount
- View and hide cart using a cart icon.
- Overlay effect when cart is open for focus.

---

## 💻 Technologies Used

- **HTML5** – For page structure
- **CSS3** – For styling and layout
- **JavaScript (Vanilla)** – For interactivity and calculations
- **Font Awesome** – For cart icon
